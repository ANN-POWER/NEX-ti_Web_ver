# NEX-ti for NEX2Y (Web)

欢迎来到 NEX-ti for NEX2Y 网页版。

## 项目结构

- `index.html`：页面结构（开始页、答题页、结果页）。
- `style.css`：网页版样式。
- `script.js`：页面交互主入口（UI流程控制）。
- `js/data/questions.js`：题库与抽题逻辑。
- `js/data/results.js`：结果文案配置。
- `js/core/scoring.js`：计分与结果判定逻辑。
- `js/core/member-bars.js`：成员成分柱图计算。
- `js/core/audio.js`：背景音乐随机播放逻辑（点击开始后触发）。
- `img/`：页面图片资源。

## 本地运行

直接在浏览器打开 `index.html` 即可。

## 说明

- 该仓库已切换为纯网页版本，不再包含小程序运行所需配置与页面文件。
